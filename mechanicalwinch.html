Project 1.2.5 - Mechanical Winch

IED - November 4, 2015

The objective of this project was to use design and build a prototype of a mechanical winch to lift a heavy weight up by three stories. I designed the winch and helped build and test it. Through this project, I learned how to effectively use sketches to plan a solution and how to improve upon it. Some challenges I faced during this project included low resource availability and faulty wires.
