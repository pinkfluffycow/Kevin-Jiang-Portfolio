Project 2.7 - Puzzle Cube

IED - November 4, 2015

The objective of this project was to use Fine Office Furniture, Inc.'s leftover wood to create a challenging puzzle cube. I was the sole designer and engineer for this project, so I designed and built the cube. Through this project, I learned how to effectively use sketches to plan a solution and how to present a solution to a wide audience. Some challenges I faced during this project included making sure that my puzzle cube satisfied the constraints specified by the client.
