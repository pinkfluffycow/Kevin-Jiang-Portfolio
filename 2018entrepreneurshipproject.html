2018 Entrepreneurship Project - Dextra

Computer Integrated Manufacturing - May 21, 2018

The objective of this project was to create an affordable, portable desk extender that would be sturdy, flexible with different desk thicknesses, and easy to install and remove. As the only person working on this project, I designed the solution, created an Inventor model for my product, and built a small proof-of-concept as well as a prototype. Through this project, I improved my skills with Autodesk Inventor's presentation features and learned about making custom textures in Inventor. Some challenges I had during this project were learning how to create custom textures as well as creating a prototype with limited materials.
